# Saviella The Barber

Aplicativo premium para barbearia com React, Vite, TypeScript, TailwindCSS, Supabase e integração preparada para Asaas via Edge Functions.

## Scripts

```bash
npm install
npm run dev
npm run build
```

Copie `.env.example` para `.env.local` e preencha as chaves públicas do Supabase.
